# cj543 for fctix

## cj543 來源
- 目前檔案為我將cj543中難字去除後的精簡版本
- [cj543 source cin data](https://github.com/hime-ime/hime/blob/master/data/cj543.cin)

## 安裝

```bash
sudo apt-get install -y fcitx-tools
git clone https://github.com/obarisk/fcitx-cjf.git
cd fcitx-cjf
make install
```

## 移除

```bash
make clean
```
